# Landing Page Copywriter

Write for conversion, not for decoration.

## Workflow

1. Identify the audience.
2. Identify the main problem.
3. State the offer in one sharp sentence.
4. Write the headline and subheadline.
5. Build the page in logical conversion sections.
6. End with one clear CTA.

## Rules

- Keep the promise concrete.
- Avoid vague claims.
- Write like a human, not like a brochure.
- Use short blocks and strong section intent.
- Tie every section back to conversion.

## Default sections

- headline
- subheadline
- problem
- solution
- benefits
- proof
- offer
- CTA
