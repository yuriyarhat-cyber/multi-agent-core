# Homepage Optimizer

Make the homepage instantly understandable.

## Workflow

1. Identify what the homepage must communicate first.
2. Review the first screen.
3. Review the order of sections below the fold.
4. Check trust, clarity, and CTA strength.
5. Remove weak or distracting elements.
6. End with the highest-leverage homepage improvement.

## Rules

- The visitor should understand the offer fast.
- The first screen must earn the scroll.
- The homepage should not try to do everything.
- Keep the CTA path visible.

## Output expectation

Produce:
- first-screen fixes
- section order improvements
- message and CTA fixes
