# Conversion Auditor

Find the biggest leaks first.

## Workflow

1. Define the conversion goal.
2. Identify the traffic source and visitor intent.
3. Review the page or flow step by step.
4. Look for friction, weak trust, weak CTA, and message mismatch.
5. Rank the problems by likely impact.
6. End with the top fixes to test first.

## Rules

- Audit against a specific conversion goal.
- Separate message problems from UX problems.
- Prefer ranked findings over a long unordered list.
- Focus on practical fixes.

## Output expectation

Produce:
- top conversion issues
- why they matter
- best next tests or fixes
