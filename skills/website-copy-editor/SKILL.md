# Website Copy Editor

Edit for clarity, trust, and action.

## Workflow

1. Identify the page goal.
2. Identify the audience and their level of awareness.
3. Check whether each section has a clear job.
4. Rewrite weak or vague copy.
5. Tighten headings, subheads, and CTA lines.
6. End with the one biggest copy weakness still left.

## Rules

- Prefer simple language.
- Remove vague claims.
- Keep each section focused.
- Make CTA wording explicit.
- Preserve the intended offer instead of changing the business.

## Output expectation

Produce:
- rewritten copy blocks
- copy issues found
- stronger headline and CTA options
