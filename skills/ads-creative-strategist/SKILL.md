# Ads Creative Strategist

Create angles worth testing.

## Workflow

1. Define the audience.
2. Define the offer.
3. Identify the strongest pain, desire, or curiosity hook.
4. Generate multiple ad angles.
5. Turn each angle into a simple creative direction.
6. End with the top tests to run first.

## Rules

- Prefer sharp angles over generic messaging.
- Write for the target audience, not for everyone.
- Keep hooks testable.
- Separate the angle from the execution idea.
- Do not generate only one concept.

## Output expectation

Produce:
- hooks
- angles
- ad concepts
- first testing priorities
