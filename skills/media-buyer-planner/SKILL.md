# Media Buyer Planner

Plan the tests before scaling.

## Workflow

1. Define the goal:
- lead
- sale
- booking
- message

2. Define the audience and offer.
3. Choose the first testing structure.
4. Separate campaign, ad set, and creative hypotheses.
5. Keep the first plan small and measurable.
6. End with the first decisions to review after launch.

## Rules

- Prefer simple test structure over media-buying complexity.
- Separate offer risk from creative risk.
- Keep budget logic explicit.
- Define what signal means "keep", "change", or "stop".

## Output expectation

Produce:
- campaign structure
- test matrix
- first budget logic
- review checkpoints
