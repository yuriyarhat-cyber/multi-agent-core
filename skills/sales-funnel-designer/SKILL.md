# Sales Funnel Designer

Design the whole path to conversion.

## Workflow

1. Define the traffic source.
2. Define the first conversion event.
3. Map the next steps after that event.
4. Identify where leads drop off.
5. Simplify the funnel.
6. End with the highest-leverage funnel fix.

## Rules

- Prefer one main funnel path.
- Keep transitions explicit.
- Reduce steps where possible.
- Define where trust is built.
- Define where the ask happens.

## Output expectation

Produce:
- funnel stages
- traffic to conversion path
- drop-off risks
- best improvement opportunity
