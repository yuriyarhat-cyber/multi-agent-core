# SEO Page Brief Writer

Plan the page before writing the page.

## Workflow

1. Define the page topic.
2. Define the search intent.
3. Define the target reader and expected answer.
4. Outline the key sections and heading direction.
5. Identify related angles the page should cover.
6. End with the biggest content gap to avoid.

## Rules

- Focus on one page intent.
- Do not mix unrelated intents on one page.
- Keep the brief practical for a writer or editor.
- Prefer clear coverage over keyword stuffing.

## Output expectation

Produce:
- page intent
- topic scope
- heading structure
- content brief
