# follow-up-sequence-writer

Use this skill to create concise follow-up sequences that keep momentum without sounding repetitive or needy.

## Workflow

1. Define the stage of the lead and the reason for follow-up.
2. Decide the length of the sequence.
3. Write 3-5 follow-up messages with different angles.
4. Set recommended spacing between messages.
5. End with a clear stop or continue condition.

## Rules

- Each message must have a distinct purpose.
- Avoid nagging or repeating the same wording.
- Keep messages short and easy to send.
- Match tone to the relationship and channel.

## Output expectation

Return:
- the full follow-up sequence
- purpose of each message
- recommended timing between touches
