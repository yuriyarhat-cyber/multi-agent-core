# sales-script-writer

Use this skill to write sales scripts that sound human, clear, and usable in real conversations.

## Workflow

1. Identify the audience, context, and channel.
2. Define the single goal of the script.
3. Write the opening and positioning.
4. Add the core message, questions, or pitch.
5. End with a clear next action or CTA.

## Rules

- Use natural language, not robotic persuasion.
- Keep one primary goal per script.
- Avoid hype, fluff, and fake urgency.
- Make the next step obvious.

## Output expectation

Return:
- the main script
- 2 short variants if useful
- the strongest CTA line
